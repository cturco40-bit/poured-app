# Poured

SaaS platform for beverage trade events. Connects Event Hosts, Agents/Producers, and Attendees.

- Live site: [pouredevents.com](https://pouredevents.com)
- App: [app.pouredevents.com](https://app.pouredevents.com)

The app is a single self-contained `index.html` — no build step. See [CLAUDE.md](./CLAUDE.md) for the full project brief.
