How it works

How a Poured event
runs end to end.

1
Event Host creates the event
Sets ticket tiers and pricing, invites agents by email, sets the wine submission deadline and door passcode
2
Agents upload their portfolios
Each agent uploads via CSV. Every supplier in the file becomes a table on the event floor. The event host reviews and approves.
3
Buyers check in and browse
Buyers enter the event with their ticket passcode, browse every table on their phone, and submit orders directly to the agent
4
Agents close the orders
Unlock buyer contact details, confirm orders, and send invoices. The event host receives their payout summary.

Is there a setup fee?
No setup fee, no monthly subscription. The $1.75 + 6% ticket fee is collected from the buyer at the point of purchase. Event Hosts receive the full face value of their tickets.
How do agents get paid?
Poured delivers the order and buyer contact details to the agent. The agent pays a 1.25% unlock fee (capped at $50) to access that contact information. Payment for wine is negotiated and processed directly between agent and buyer, outside of Poured.
Do I need a contract?
No contract required. Create an account, set up your first event, and pay only when tickets sell. You can run your next event the same day.
